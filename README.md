# Lê Duy Quân — Portfolio

Modern, bilingual portfolio built with semantic HTML, CSS and vanilla JavaScript.

## Run locally

Open `index.html` directly, or serve the folder with any static server:

```bash
npx serve .
```

## Deploy

The site is fully static and can be deployed to GitHub Pages, Cloudflare Pages, Netlify or Vercel without a build step.
